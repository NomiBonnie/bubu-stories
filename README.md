# 咘咘的故事书 🐰

A simple, mobile-friendly picture book reader for Bubu, built with Vite + React.

## Live Site
👉 **https://nomibonnie.github.io/bubu-stories/**

## Features
- 📖 Slide-style reader (swipe or click to turn pages)
- 🎨 Pixar 3D-style illustrations generated by Azure GPT-image-1.5
- 🎵 Chinese TTS narration (Tingting voice)
- 📱 Mobile-optimized for toddlers
- 🌙 Story: "小兔子找月亮" (Little Bunny Looking for the Moon)

## Development
```bash
npm install
npm run dev      # local dev server
npm run build    # build for production
npm run deploy   # deploy to GitHub Pages
```

## Story Data
Stories are stored in `stories/YYYY-MM-DD.json` with text and image prompts.

## Tech Stack
- Vite 8 + React 19
- Azure OpenAI GPT-image-1.5 (medium quality, 1536×1024)
- macOS `say` command (Tingting voice) → ffmpeg → MP3
- gh-pages deployment
