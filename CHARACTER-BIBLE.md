# 咘咘绘本 · 角色圣经 (Character Bible)

> **铁规：每次生成插画前必须读这个文件，每张图的 prompt 必须内联角色完整描述。**
> **新角色出现后立即录入此文件，哪怕只是配角。**

## 🌟 主角

### 咘咘 / 布布 (Bubu)
- **物种**：兔子 🐰
- **毛色**：纯白色（100% snow-white fur），绝对不能偏灰/米黄/奶油色
- **耳朵**：长耳朵，内侧粉色
- **眼睛**：大圆眼睛，棕色/深色瞳孔，闪亮
- **鼻子**：小粉色鼻子
- **服装**：粉红色连衣裙 + 粉色小蝴蝶结（头顶或耳朵上）
- **体型**：幼儿比例，圆润可爱
- **备注**：Story 1-6 早期版本没穿裙子，Story 7+ 统一穿粉裙子

### NOMI
- **物种**：浣熊 🦝
- **毛色**：标准浣熊色（灰棕色，黑色眼罩纹路，环纹尾巴）
- **眼睛**：大圆眼睛，聪明灵动
- **服装**：蓝色条纹毛衣（蓝白横条纹）
- **配饰**：偶尔戴眼镜（看书时）
- **体型**：比咘咘略大，灵巧手爪
- **性格外观**：好奇、温柔、手巧

### NONO
- **物种**：小鸟 🐦
- **羽毛**：红色（鲜红色羽毛，不是橙红）
- **体型**：小巧，站在角色肩膀或头顶
- **眼睛**：圆亮小眼睛
- **喙**：橙黄色小喙
- **备注**：飞行姿态时翅膀展开

## 📐 通用规则

1. **画风**：Pixar 3D style，温暖柔和的光线，儿童绘本质感
2. **尺寸**：竖版 1024×1536
3. **每张图 prompt 必须内联角色描述**，不能只写名字
4. **白色 = 纯白**，不能是米白/奶油色/灰白
5. **新角色首次出现时，立即更新本文件**
6. **跨故事复用角色时，必须查本文件确认外观**
